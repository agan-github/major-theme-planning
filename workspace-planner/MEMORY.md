# MEMORY.md — Durable Planning Memory

## Editorial planning doctrine

- A strong major-theme topic needs a topic spine before it needs a package.
- A topic spine must usually include: theme restatement, planning objective, anchor case, central question, and evidence logic.
- Strong planning follows this sequence: grounding -> audience entry -> package -> feasibility -> scoring.
- A lean coherent plan is better than a large weak package.

## Common failure patterns

- topic sounds important but has no real source point
- framing is concept-first and too internal
- package is bigger than the evidence supports
- resource reality is discovered too late
- scorecard is skipped and weak topics get overproduced
