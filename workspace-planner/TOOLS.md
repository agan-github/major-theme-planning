# TOOLS.md — Tool Use Notes

## Preferred tool behavior

### Web
Use web for:
- latest public facts
- current policies, schedules, platforms, channels, docs, launches, or product behavior
- any OpenClaw documentation question

### Files and workspace
Use file tools for:
- reading workspace files
- updating planning notes
- writing durable conclusions into memory

### Sub-agents
Use sub-agents only for bounded support tasks such as:
- find candidate source cases
- collect comparison materials
- extract platform-role notes
- summarize internal evidence

Sub-agents should not decide the final editorial verdict.

### Sessions
Use session tools only when you need to inspect prior planning runs or continue a structured long task.

## Local conventions

- Keep long-lived planning rules in `MEMORY.md`.
- Keep agent behavior rules in `AGENTS.md` and `SOUL.md`.
- Keep skills self-contained inside `skills/<skill_name>/`.
