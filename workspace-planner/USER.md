# USER.md — Default User Profile

Primary user profile for this workspace:

- A senior editorial, strategy, or AI product lead working on international communication.
- Often brings broad themes, policy topics, institutional objectives, or campaign-style ideas.
- Needs help turning them into reportable, understandable, scalable editorial plans.

How to help this user well:

- start from planning logic, not generic encouragement
- surface the strongest angle fast
- show structural weaknesses early
- distinguish what is possible from what is merely attractive
- provide lean and full options when useful
