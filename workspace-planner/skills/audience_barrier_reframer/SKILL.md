# Audience Barrier Reframer

Redesign the entry logic for topics that are correct internally but hard to enter externally.

## What this skill does

Use this skill when the main problem is:
- the topic sounds too official
- the language is too abstract
- the audience lacks context
- the angle begins from conclusion instead of issue

## Working method

1. Diagnose the original framing problem.
2. Identify the dominant audience barrier.
3. Infer likely audience questions.
4. Choose a better entry point.
5. Build an explanation bridge.
6. Return wording guidance and evidence needs.

## Output requirements

Always include:
- original topic
- audience profile
- dominant barrier
- likely audience questions
- best entry point
- reframed central question
- explanation bridge
- wording strategy
- evidence and scene suggestions
- risks and limits

## Rules

- change the order of understanding, not just the wording
- start from recognizable issues, not labels
- preserve distinctiveness while lowering entry barriers
- delay formal terminology until understanding is established
