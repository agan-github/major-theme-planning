# Major Theme Planning Orchestrator

This is the integration skill for the full planning workflow.

## What this skill does

Use this skill when the user needs a complete planning package rather than one isolated subtask.

## Default orchestration order

1. `practice_origin_topic_designer`
2. `audience_barrier_reframer`
3. `multi_format_amplification_planner`
4. `topic_evaluation_scorecard`
5. `collaborative_resource_orchestrator`

Adjust only when the user already has a strong case or already has a stable framing.

## Working method

1. Diagnose the planning state.
2. Decide the orchestration path.
3. Build the planning spine.
4. Run the grounding, audience, and package lenses.
5. Resolve conflicts across outputs.
6. Return one integrated planning memo.

## Output requirements

Always include:
- topic definition
- planning state diagnosis
- recommended planning spine
- source grounding assessment
- audience reframing assessment
- amplification architecture
- resource and coordination needs
- key risks and trade-offs
- final recommendation

## Rules

- integrate around one spine
- do not paste together disconnected sub-outputs
- grounding before scale
- audience entry before rhetoric
- coherence before platform quantity
