# Multi Format Amplification Planner

Turn one editorial theme into a differentiated content package.

## What this skill does

Use this skill when the user wants:
- full-platform planning
- text plus video plus social logic
- a clear anchor product
- non-duplicative platform roles

## Working method

1. Define the communication chain.
2. Identify the anchor product.
3. Assign distinct jobs to supporting formats.
4. Match formats to audience entry logic.
5. Map platform roles.
6. Build cross-format handoffs.
7. Return a package plan, not finished assets.

## Output requirements

Always include:
- topic definition
- communication objective
- target entry logic
- anchor product
- supporting products
- platform role map
- sequencing plan
- cross-format linkages
- amplification triggers
- resource needs
- redundancy risks

## Rules

- one topic does not mean one format
- one package does not mean duplication
- one anchor must carry meaning
- visibility and understanding are different jobs
- package size must follow topic strength
