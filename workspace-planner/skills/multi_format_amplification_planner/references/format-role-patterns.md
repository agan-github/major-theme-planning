# Format Role Patterns

## Main communication jobs

- anchor meaning
- create first visibility
- lower entry barriers
- visualize proof
- humanize the topic
- generate shareable fragments
- sustain attention
- reactivate deeper engagement

## Strong package signals

- one clear anchor product
- differentiated supporting formats
- explicit platform jobs
- audience progression from notice to understanding
- realistic scope

## Weak package signals

- same content repeated everywhere
- no anchor product
- every asset tries to explain everything
- strong spread but weak meaning control
