# Topic Evaluation Scorecard

This skill is the editorial judgment layer.

## What this skill does

Use this skill when you need to:
- assess one topic
- compare several topics
- decide whether a topic should scale
- identify repair priorities

## Seven dimensions

1. source grounding
2. audience entry
3. distinctiveness
4. evidence and scene strength
5. execution feasibility
6. amplification potential
7. strategic fit

## Output requirements

Always include:
- topic definition
- evaluation scope
- seven-dimension scorecard
- strength summary
- structural weaknesses
- scale recommendation
- repair priorities
- final verdict

## Rules

- score structure before enthusiasm
- fatal weaknesses matter more than decorative strengths
- strategic importance is not enough without a usable angle
- accessibility is not enough without distinctiveness
- scale must be earned
